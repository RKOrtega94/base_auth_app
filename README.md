## Getting Started

### 1. Clone the repo

```sh
git clone https://github.com/RKOrtega94/base_auth_app.git 'project_name'
```

### 2. Install dependencies

```sh
flutter pub get
```

### 3. Create .env file and replace the values

```sh
cp .env.example .env
```

### 4. Run the app

```sh
flutter run
```
